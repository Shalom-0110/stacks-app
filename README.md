# Stacks 💸

Your everyday financial companion. Spend smarter, save consistently, invest confidently.

A personal finance app built for young professionals and early adults. Think CRED meets Duolingo, not QuickBooks.

---

## What's Inside

### V1 - Live Now
- **Home Dashboard** - Financial health score, income/spend/savings summary, spending breakdown, recent transactions
- **Spend Tracker** - Log expenses by category, search and filter, monthly summary
- **Earn** - Income tracking across Salary, Freelance, Side Hustle, Gifts, Refunds
- **Goals** - Savings goals with circular progress, deadline countdowns, completion celebrations
- **Budget Planner** - Per-category limits, color-coded progress bars, overspend alerts
- **Subscription Tracker** - All recurring bills, renewal countdowns, annual cost projection
- **Quick Add** - Global modal to log any expense in seconds from anywhere
- **Financial Health Score** - Composite score shown across the app

### V2 - Coming Next
Investments, Learning Hub, Challenges and Gamification, Wishlist Planner, Financial Calendar, Smart Insights

### V3 - Future
AI Money Coach, Bank/UPI Sync, SMS Expense Detection, Receipt Scanning, Portfolio Tracking

---

## Tech Stack

- **Frontend:** Vue 3, Vue Router, Vuex
- **UI:** PrimeVue 4, Tailwind CSS 4
- **Charts:** ApexCharts
- **Backend:** Django 5.2
- **Build:** Vite 6

---

## Getting Started

```bash
# Frontend
npm install
npm run dev

# Backend
pip install -r requirements.txt
python manage.py runserver
```
